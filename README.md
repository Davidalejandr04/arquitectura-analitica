# Arquitectura analítica

Este proyecto simula el diseño y la implementación de una arquitectura analítica modular siguiendo principios estructurados y escalables.

La solución incluye un enfoque en capas con ingesta de datos, validación, modelado relacional y procesamiento analítico. Si bien se implementa localmente, el diseño sigue las mejores prácticas comúnmente utilizadas en plataformas analíticas en la nube
